# Page-level IA — Category content matrix (node 2.x · data)

- **What it is:** the single structured artifact that **populates every category** that runs on
  the shared listing template (`category.md`, node 2.1). One row per category → its facet subset,
  relevant brands, goal links, SEO meta (Title/Description/H1), ready SEO body text, FAQ, and
  related categories.
- **Why one artifact:** brands and facets are largely **shared** across categories (same product
  pool). A single matrix dedupes that (shared brand pool + global facet definitions) and stays the
  one source of truth, instead of 12 separate page specs.
- **Canonical visual:** `ia/category-matrix.html` (Ukrainian). This markdown is the source of truth.
- **Drives:** the per-category instance of the listing (2.1), the «Каталог» mega-menu, goal pages
  (2.2), and the footer SEO links (0.2). Taxonomy lives in `catalog.md`; this fills it with content.
- **Scope:** 11 listing categories (Бренди = a parallel index 2.4, not a content row).

## Shared brand pool (master list — real, UA-available)

Optimum Nutrition · BioTechUSA · Myprotein · Scitec Nutrition · Dymatize · BSN · MuscleTech ·
Universal Nutrition · Weider · OstroVit · Olimp Sport Nutrition · Trec Nutrition · Maxler ·
Quamtrax · Applied Nutrition · NOW Foods · Solgar · Puritan's Pride · Power Pro (UA) ·
Sporter (UA) · AB PRO (UA) · Progress Nutrition (UA) · BlenderBottle · SmartShake.

Per-category «brands» below is a **subset** of this pool. The brand facet is the same component
everywhere; only the values + counts differ. Exact catalog depth stays the operational `[?]`.

## Global facet definitions (the inherited filter panel)

Тип/субкатегорія · Наявність · Ціна · Бренд · Ціль · Форма · Смак · Фасування/вага · Країна ·
Сертифікація — plus category-specific facets noted per row (e.g. «Вміст кофеїну», «Без лактози»,
«Об'єм» for shakers). Each category shows only the facets that make sense for it.

## SEO meta rules (apply to every row)
Title ≤ 60 chars · Description ≤ 155 · single H1 = the category name · canonical to the clean
category URL · paginated pages self-canonical · hreflang uk/ru/x-default · BreadcrumbList +
ItemList/CollectionPage + optional FAQPage. Faceted-nav indexation: only the curated whitelist
(brand-in-category, goal-in-category, city 2.1a) indexed; other combos canonical-to-base.
