Спортивне харчування в Україні — Stack

Stack — інтернет-магазин спортивного харчування з доставкою по всій Україні. У каталозі — протеїн, гейнери, креатин, амінокислоти (BCAA, EAA), передтренувальні комплекси, жироспалювачі, вітаміни та продукти для здоров'я від перевірених брендів. Уся продукція сертифікована, з оригінальною гарантією виробника.

Не знаєте, з чого почати? Оберіть ціль — набір маси, схуднення, відновлення, енергія, імунітет чи витривалість — і отримайте зрозумілий, безпечний набір із вказаним складом і дозуванням. Швидка доставка Новою Поштою за 1–2 дні, зручна оплата карткою чи готівкою та повернення протягом 14 днів.

Текст лагідний і інформативний (Принцип №5), з кореневими ключами без переспаму. Популярні запити-лінки (категорії/бренди/міста) — у футері (0.2).

E · Чек-лист оптимізації

  • Один H1, логічна ієрархія H2 → H3; кожна секція має змістовний заголовок.
  • LCP-банер оптимізований і не блокує рендер; зображення з alt, нижні — lazy-load.
  • Внутрішня перелінковка крауленими <a> (плитки цілей і категорій) з описовими анкорами.
  • canonical + hreflang (uk/ru), без дублів контенту.
  • Смуга довіри — текст, не картинки (читається краулером).
  • Core Web Vitals у нормі; мова контенту — українська.

13 · Доступність

Доступність

SEO винесено в окремий блок (08). Тут — доступність.

Логічний порядок заголовків H1 → H2; кожна секція має заголовок.
Плитки — тап-цілі ≥ 44px, видимий фокус.
Каруселі товарів — керування клавіатурою, кнопки «вперед/назад» з aria-label.
Полиця кошика / персональна стрічка — оголошення лічильника й суми (aria-label).
Контраст тексту на банерах ≥ 4.5:1 (перевірити на етапі кольору).

Рішення зафіксовано ✓

2026-07-01 — структура головної (Foxtrot-патерн):

  • Перший екран = рейл категорій + банери (Foxtrot/Rozetka): лівий рейл — «За ціллю» першим пунктом (концерн-лінза, як у каталозі), далі 12 категорій (hover → флайаут) + банер-слайдер 2–3 картки + 2 вертикальні промо (опт-банер «Для тренерів» — лише тренерам).
  • Полиця кошика (стан) одразу під hero, коли кошик не порожній: N товарів, сума, мініатюри, «Оформити замовлення» / «Переглянути кошик». Персональна стрічка за роллю — варіант цього блоку.
  • «Спортивне харчування під вашу ціль» (блок 3) — великий хедінг + опис (як було) + 6 плиток + CTA «Пройти квіз» (не «Відкрити каталог»). Front-door новачка. Окремий блок «популярні категорії» прибрано (рейл покриває).
  • Квіз більше не губиться: головний CTA блоку цілей + кнопка «Квіз» у хедері замість мега-меню «Цілі» (цілі вже добре показані в каталозі: 2.0/2.1/2.2). Квіз (4.x) післязапуск; MVP → плитки цілей.
  • Банер-слайдер = 2–3 картки (не одна) + вертикальні промо; опт-банер «Для тренерів» таргетований лише на тренерів (опт-ціни не показуємо всім). Окремий блок «Для тренерів» лишається (Принцип №3).
  • Товари — тематичні добірки-таби: Хіт продажу · Кращі цінові пропозиції · Преміум товари · Новинки (+ Переглянуті — стан).
  • Канонічна картка (Foxtrot/Belok), порядок: вертикальне фото (зменшене, ♡ + кутовий Хіт/NEW) → назва до 2 рядків → бренд · країна → ціна (нова + стара з −% поруч) + іконка-кнопка 🛒 збоку → рейтинг + відгуки (після ціни) → мета-рядок ціна/порція + бонус. Канонічна → прокинути в лістинги (2.1/2.0).
  • SEO-блок (A–E) повний ще на IA — шаблон для кожної сторінки. Смуга довіри високо (Принцип №1). Видимий «Для тренерів» (Принцип №3) → 5.0.
  • Спокійно, без countdown-таймерів (Принцип №4). Мокапи ч/б; колір — на концепті. Порядок продуманий з мобілки (рейл → банер + чипи).

Відкриті [?]: правила добірок (хіти/новинки/рекомендації) і добірка банерів — дані продажів/маркетинг; ціна за порцію + бонус — реальні значення на етапі даних; квіз (4.x) і «Мої стейпли» — післязапуск; чи показувати «переглянуті» гостю.
